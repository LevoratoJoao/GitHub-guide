Atalhos uteis no geral
ctrl+a seleciona o código todo
backspace apaga o caracter que esta a direita do cursor
del apaga o caracter que esta a esquerda do cursor
ctrl+del apaga a palavra que esta a direita do cursor
ctrl+backspace apaga a palavra que esta a esquerda do cursor
home move o cursor para o inicio da linha
end move o cursor para o fim da linha
shift+home seleciona o texto da posição do cursor até ao inicio da linha
shift+end seleciona o texto da posição do cursor até ao fim da linha
ctrl+home move o cursor para o inicio do código
ctrl+end move o cursor para o fim do código
ctrl+shift+home seleciona o texto da posição do cursor até ao inicio do código
ctrl+shift+end seleciona o texto da posição do cursor até ao fim do código
page up pula equivalente a uma tela para cima
page down pula equivalente a uma tela para baixo
ctrl+shift+page up pula equivalente a uma tela para cima e seleciona o texto
ctrl+shift+page down pula equivalente a uma tela para baixo e seleciona o texto
ctrl+z desfaz a ultima modificação feita
ctrl+y refaz a ultima modificação feita
ctrl+x recorta o texto selecionado (copia mas apaga o que foi copiado)
ctrl+⇦ vai para o começo da palavra onde esta o cursor
ctrl+⇨ vai para o final da palavra onde esta o cursor
shift+⇦ seleciona o caracter a esquerda do cursor
shift+⇨ seleciona o caracter a direita do cursor
ctrl+shift+⇨ seleciona a palavra a direita do cursor
ctrl+shift+⇦ seleciona a palavra a direita do cursor
ctrl+f pesquisa o que desejar no arquivo
ctrl+s salva o arquivo
ctrl+= aumenta zoom
ctrl+- diminui zoom
Mouse clicks
dois clicks na palavra seleciona a palavra
três clicks na linha seleciona a linha
Atalhos uteis no vscode
ctrl+shift+p Abre a barra de comandos
Edição básica
alt+⇧/⇩ Move a linha para cima/baixo
shift+alt+⇧/⇩ Copia linha para cima/baixo
ctrl+shift+k excluir linha
ctrl+ENTER Insere linha abaixo
ctrl+shift+ENTER Insere linha acima
ctrl+⇧/⇩ Scroll linha para cima/baixo
ctrl+/ Adiciona/remove comentario
shift+tab recua linha
ctrl+[ avança linha (tab)
Navegação
F8 Vai para proximo erro/warning
shift+F8 Vai para o erro/warning anterior
ctrl+shift+tab Navega pelo historico de arquivos
Pesquisa e replace
ctrl+h Abre a barra de replace
ctrl+d seleciona proxima ocorrencia da palavra onde o cursor esta
Multi-cursor e seleção
alt+click Insere cursor
ctrl+alt+↑/↓ Insere cursor acima/abaixo
ctrl+u Desfaz a ultima seleção
ctrl+l seleciona a linha atual
ctrl+F2 seleciona todas as ocorrencias da palavra onde o cursor esta
Edição de linguagens
shift+alt+f formata o codigo
ctrl+f ctrl+f formata o que esta selecionado
F12 Vai para a definição da função
ctrl+. Fix rápido
F2 Renomeia
Gerenciamento de editor
ctrl+w/ctrl+F4 Fecha o arquivo aberto
ctrl+\ divide o editor em 2
ctrl+1/2/3 alterna entre os editores criados
Gerenciamento de arquivos
ctrl+n Abre um novo arquivo
ctrl+o Abre um arquivo existente
ctrl+tab Abre a barra de seleção de arquivos
Exibição
ctrl+b Mostra/Oculta a barra lateral
ctrl+shift+e Mostra/Oculta o explorer
ctrl+shift+f Mostra/Oculta a barra de pesquisa
ctrl+shift+g Mostra/Oculta o git
ctrl+shift+d Mostra/Oculta o debug
ctrl+shift+x Mostra/Oculta as extensões
debug
F5 Inicia o debug
shift+F5 Para o debug
F11 Entra na função
F10 Passa para a proxima linha sem entrar na função
Terminal integrado
ctrl+j Abre o terminal
ctrl+click em uma chamada de função te leva até a definição dela
